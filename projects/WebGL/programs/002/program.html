Program 2

Simple shape generator. There are two variants:

The goal of this program is to learn to make shapes in WebGL using coordinate calculations.

Object 1: Starts off with a triangular polygon, increasing the number of sides changes the polygon. Can go on indefinitly till it appears as a circle. You can also change the pivot point and shading intensity for different effects.

Object 2, generates a star shape that changes the color and lines randomly after pressing 'Object 2' each time.
